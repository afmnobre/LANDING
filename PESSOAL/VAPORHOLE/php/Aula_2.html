Tipos de Operadores de Comparação

Exemplo:


$a = 5;
$b = 3;
$c = 0;

OperadorNomeExemploResutado
==Igualif($a == $b)false
===Identicoif($a === $b)false
!=Diferenteif($a != $b)true
<>Diferenteif($a <> $b)true
!==Não Identicoif($a !== $b)true
>Maior queif($a > $b)true
<Menor queif($a < $b)false
>=Maior ou Igualif($a >= $b)true
<=Menor queif($a <= $b)false
<=>Spaceshipif($a <=> $b)1
??Diferente (null coalescing)if($a ?? $b ?? $c)5

Spaceship - regras:

Retorna = 0 : Se os valores de ambos os lados forem iguais.
Retorna = 1 : se o valor à esquerda é maior.
Retorna = -1 : se o valor à direita é maior.

Null Coalescing - regras:

Avalia as variáveis da esquerda pra direita e o primeiro valor não NULL sera resultado da Comparação.

Estrutura de Controle IF, ELSE e ELSEIF

Exemplo 1:


if ( $a > $b ){
	echo "A é maior que B";
} elseif ( $a == $b )
	echo "A é igual a B";
} else {
	echo "A não é maior que B";
}

Exemplo 2:


$idade = 17;

if ( $idade >= 18 ){
	echo "Pode Tirar Habilitação";
} elseif ( $idade >= 16 ){
	echo "Pode Votar mas não tirar a Habilitação";
} else {
	echo "Não pode votar nem tirar Habilitação";
}

Resultado: Pode Votar mas não tirar a Habilitação.

Operador Ternário

Exemplo 1:


$a = 40;

$resultado = ( $a >= $b ? "Maior ou Igual a B" : "Menor que B" );
echo $resultado;

Exemplo 2:


$valor = 100;

$aprovado = ( $valor >= 100 ) ? true : false;

Operador de Incremento e Decremento

Exemplo 1:


$x = 10;
$y = 10;

Pré incremento:
echo "Valor de X".++$x;
Resultado: Valor de X 11
echo "Valor de X".$x;
Resultado: Valor de X 11

Pré decremento:
echo "Valor de Y ".--$y;
Resultado: Valor de Y 9
echo "Valor de Y ".$y;
Resultado: Valor de Y 9

Pós incremento:
echo "Valor de X".$x++;
Resultado: Valor de X 10
echo "Valor de X".$x;
Resultado: Valor de X 11

Pós decremento:
echo "Valor de Y ".$y--;
Resultado: Valor de Y 10
echo "Valor de Y ".$y;
Resultado: Valor de Y 9

Estrutura de Controle - DO WHILE

Exemplo DO WHILE:


$x = 1;

do {
	echo "O Número é: $x";
	$x++;
} while ($x <= 5);

O Número é: 1
O Número é: 2
