Instalação do Nvim e Vim

0.1) Necessário Instalar o Python para funcionamento de futuros Plugins

(DEBIAN)

sudo apt update
sudo apt install python3-pip

O Pip 3.9 pode ser instalado assim:

curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py
sudo python3.9 get-pip.py

0.2) Biblioteca pynvim (Para fazer o Neovim utilizar o Python).

pip install pynvim

0.3) Instalação do Nodejs

Executar os procedimentos do link abaixo para instalar o Nodejs

https://github.com/nvm-sh/nvm#installing-and-updating

Depois reinicie o terminal e executar o seguinte comando:

nvm install –lts

1.0) Instalação do VIM (Linux Debian e WSL)

sudo add-apt-repository ppa:jonathonf/vim
sudo apt update
sudo apt install vim

1.1) Instalação do NVIM(Linux Debian e WSL)

sudo add-apt-repository ppa:neovim-ppa/unstable
sudo apt-get update
sudo apt-get install neovim

1.2) Instalação VIM(Arch Linux)

sudo pacman -S vim

1.3) Instalação NVIM(Arch Linux)

sudo pacman -S neovim

Arquivos de Configuração

2.1) VIM

Criar no diretório ~/ o arquivo oculto .vimrc

2.2) NVIM

Criar no diretório ~/.config/ um diretório chamado nvim e depois um arquivo chamado init.vim

3.1) Exemplo de conteudo de arquivo de configuração do VIM e do NVIM.

" Global Sets """""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""
syntax on            " Enable syntax highlight
set nu               " Enable line numbers
set tabstop=4        " Show existing tab with 4 spaces width
set softtabstop=4    " Show existing tab with 4 spaces width
set shiftwidth=4     " When indenting with '>', use 4 spaces width
set expandtab        " On pressing tab, insert 4 spaces
set smarttab         " insert tabs on the start of a line according to shiftwidth
set smartindent      " Automatically inserts one extra level of indentation in some cases
set hidden           " Hides the current buffer when a new file is openned
set incsearch        " Incremental search
set ignorecase       " Ingore case in search
set smartcase        " Consider case if there is a upper case character
set scrolloff=8      " Minimum number of lines to keep above and below the cursor
set colorcolumn=100  " Draws a line at the given line to keep aware of the line size
set signcolumn=yes   " Add a column on the left. Useful for linting
set cmdheight=2      " Give more space for displaying messages
set updatetime=100   " Time in miliseconds to consider the changes
set encoding=utf-8   " The encoding should be utf-8 to activate the font icons
set nobackup         " No backup files
set nowritebackup    " No backup files
set splitright       " Create the vertical splits to the right
set splitbelow       " Create the horizontal splits below
set autoread         " Update vim after file update from outside
set mouse=a          " Enable mouse support
filetype on          " Detect and set the filetype option and trigger the FileType Event
filetype plugin on   " Load the plugin file for the file type, if any
filetype indent on   " Load the indent file for the file type, if any

Instalação de VIM PLUG

4.1) Colocar as seguintes linhas no TOPO do arquivo de configuração e de acordo com seu Plugin colocar as configurações
de acordo com o solicitado.

call plug#begin()
" Plugins aqui
	<nome do Plugin 1>
	<nome do Plugin 2>
	...
call plug#end()

4.2) VIM

iwr -useb https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim |`
	ni $HOME/vimfiles/autoload/plug.vim -Force

4.3) NVIM

iwr -useb https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim |`
	ni "$(@($env:XDG_DATA_HOME, $env:LOCALAPPDATA)[$null -eq $env:XDG_DATA_HOME])/nvim-data/site/autoload/plug.vim" -Force

4.4) Fazendo esses passos o Gerenciador de Plugin ja esta pronto para instalar outros plugins com o comando:

:PlugInstall <Nome do Plugin>
  • Sempre que instalar e configurar um plugin no Arquivo de configuração, reinicie o VIM e NVIM para executar o comando acima.

4.5) Para atualizar o Plugin:

:PlugUpdate
