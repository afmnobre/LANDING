Manual de Configuração TMUX

Pagina de Download do Plugin:

https://github.com/tmux-plugins/tpm

Comando para download do Plugin:

git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm

Arquivo de configuração abaixo:

	
	unbind r
	bind r source-file ~/.tmux.conf

	set -g prefix C-s


	# Para navegar como o VIM e o Nvim.

	setw -g mode-keys vi
	bind-key h select pane -L
	bind-key j select pane -D
	bind-key k select pane -U
	bind-key l select pane -R

	# Lista de Plugins
	set -g @plugin 'tmux-plugin/tpm'
	set -g @plugin 'christoomey/vim-tmux-navigator'
	set -g @plugin 'dracula/tmux'


	# configuração do tema DRACULA do STATUS BAR.

	set -g @dracula-show-powerline true
	set -g @dracula-fixed-location ""
	set -g @dracula-plugins "weather"
	set -g @dracula-show-flags true
	set -g @gracula-show-left-icon session
	set -g status-position top

	# Inicialização do PLUGIN MANAGER ( TEM QUE FICAR NO FINAL DO ARQUIVO)
	run '~/.tmux/plugins/tpm/tpm'
